# 💱 Conversor de Moedas 
Projeto desenvolvido em **Java** como **desafio final do curso Java Consumindo APIs**, do programa **Oracle Next Education (ONE)**.

Este projeto tem como objetivo aplicar na prática conceitos de consumo de API, requisições HTTP, manipulação de JSON e organização de código utilizando Programação Orientada a Objetos (POO).

<br>

## 📌 Sobre o Projeto

O **Conversor de Moedas** é uma aplicação de console que:

- Permite ao usuário escolher moedas de origem e destino
- Realiza requisições HTTP para obter taxas de câmbio atualizadas
- Converte valores dinamicamente
- Exibe o resultado formatado no terminal

O projeto utiliza:

- `HttpClient` (Java 11+)
- Biblioteca **Gson** para manipulação de JSON

<br>

## 🚀 Tecnologias Utilizadas

- ☕ Java 11+
- 🌐 API de câmbio (Exchange Rate API)
- 📦 Gson (Google JSON Library)

## ⚙️ Como Executar o Projeto

### 1️⃣ Clone o repositório

```bash

````


## 🔑 Configuração da API

Certifique-se de configurar sua chave de API como variável de ambiente:

### Linux / Mac:

```bash
export API_KEY="sua_chave_aqui"
```

### Windows (PowerShell):

```powershell
setx API_KEY "sua_chave_aqui"
```
